  • Voxel Based Volume Renderer and Simulator

    C++ OpenGL

    In this project I was interested in different ways to display volumes made up from vexels that are stored on the gpu in the form of high resoulution textures.

    My initial idea came from a recent concept I learnt about involing rendering hair and sometimes grass on 3D models. This inspired me to layer quads together and use their positions to sample a texture with transparancy in order to from the volume.

    The volume texture itself was designed to be interated upon and perform different simulations in this space

    Ultimetly I changed the rendering method to use marching rays to remove previous artifacts and allow more interested ray based calculations.

    Later on inbetween the raytrace and displaying on screen steps was added a step involing multisampling the raytraced result and bluring the result to hide the voxels and create a cloud effect. The raytrace result resoulution could then be subsequently reduced.

      then upscale and blur the final result.

  • Javascript Raytracer

    Javascript Html Canvas-Element

    This is a project I was given during my first year of university, the primary objective was to write a render using javascript and the canvas container that uses raytracing and a simple lighting model, This means a directional light, specular reflection and shadows of spheres within a scene.

    I had gone a few extra steps and added antialiasing which would first render the image, using a kernal scan it for object's edges, re-render the edges of the objects to create softer and more accurate outlines of shapes and details.

    Further more I added the ability to draw triangles into the scene, this ment that I could then use the vertex information of these models to map a variety of textures onto the faces and ultimetly render fully textured models.
