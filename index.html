  • Compute Based terrain edtior in deferred rendered scene

    C++ OpenGL

    This project focused on using a verity of 3D rendering techniques with the final version being deferred rendered with pbr lighting and directional light shadows.

    A few of the unique features I decided to add was the editable terrain and overhang particles as well as a raycast implementation for player floor detection and editing.

    The terrain editing is handled by a compute shader that edits a displacement texture of which all the values (rgb -> xyz) are used to offset the terrain geometry.

    The overhang particles intercept the tesselated terrain geometry data in the geometry shader and write them to an SSBO, these points are then used as the starting points for the particles which are contained in another SSBO and are controlled by a compute shader.

    The raycasting is made by reading back from the buffer on the gpu to the cpu a single rendered pixel in a given direction in the case of the floor its straight down, the information I wrote to the buffer was the position relative to the viewport and the ID of the object in the object list.

  • Voxel Based Volume Renderer and Simulator

    C++ OpenGL

    In this project I was interested in different ways to display volumes made up from vexels that are stored on the gpu in the form of high resoulution textures.

    My initial idea came from a recent concept I learnt about involing rendering hair and sometimes grass on 3D models. This inspired me to layer quads together and use their positions to sample a texture with transparancy in order to from the volume.

    The volume texture itself was designed to be interated upon and perform different simulations in this space

    Ultimetly I changed the rendering method to use marching rays to remove previous artifacts and allow more interested ray based calculations.

    Later on inbetween the raytrace and displaying on screen steps was added a step involing multisampling the raytraced result and bluring the result to hide the voxels and create a cloud effect. The raytrace result resoulution could then be subsequently reduced.
